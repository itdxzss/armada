## armada schema（V005 账号块，2026-06-25）

### account（账号身份主表）

| Column | Type | Nullable | Default | Description |
|--------|------|----------|---------|-------------|
| id | BIGINT | NO | AUTO_INCREMENT | 主键 |
| tenant_id | BIGINT | NO | - | 租户ID(拦截器注入) |
| ws_phone | VARCHAR(32) utf8mb4_bin | NO | - | WA号(按字节精确去重) |
| account_type | TINYINT | NO | - | 账号类型:1个人 2商业(导入即冻结,不得改写) |
| device_os | TINYINT | YES | NULL | 机型:1安卓 2苹果 |
| number_source | TINYINT | YES | NULL | 来源:1买量 2裂变 3自购 |
| channel_name | VARCHAR(64) | YES | NULL | 推广渠道名 |
| ownership | TINYINT | NO | 1 | 归属:1自有 2平台 3租借 |
| lease_until | BIGINT | YES | NULL | 租借到期(epoch毫秒;ownership=3) |
| account_group_id | BIGINT | YES | NULL | 归一单分组(→account_group.id) |
| protocol_id | VARCHAR(32) | YES | NULL | 接入协议标识(系统分配) |
| protocol_account_id | VARCHAR(64) | YES | NULL | 协议账号句柄 acc_<wsPhone> |
| protocol_address | VARCHAR(128) | YES | NULL | 协议地址 |
| priority | INT | NO | 0 | 选号优先级 |
| dispatched_at | BIGINT | YES | NULL | 首次派单时间(epoch毫秒;step1恒NULL=未分配) |
| remark | VARCHAR(255) | YES | NULL | 备注 |
| created_at | BIGINT | NO | - | 入库时间(epoch毫秒,应用层写) |
| updated_at | BIGINT | NO | - | 更新时间(epoch毫秒,应用层写) |
| created_by | BIGINT | YES | NULL | 创建人user_id |
| deleted_at | BIGINT | YES | NULL | 软删时间(epoch毫秒);NULL=未删 |
| is_active | TINYINT VIRTUAL | YES | （生成列） | 软删唯一键辅助:活=1 删=NULL |

**Indexes:**

| Name | Columns | Type |
|------|---------|------|
| PRIMARY | id | PRIMARY |
| uq_tenant_phone | tenant_id, ws_phone, is_active | UNIQUE |
| idx_tenant_group | tenant_id, account_group_id | INDEX |
| idx_tenant_created | tenant_id, created_at | INDEX |

### account_state（账号生命周期状态，高频Kafka回写）

| Column | Type | Nullable | Default | Description |
|--------|------|----------|---------|-------------|
| id | BIGINT | NO | AUTO_INCREMENT | 主键 |
| tenant_id | BIGINT | NO | - | 租户ID |
| account_id | BIGINT | NO | - | →account.id |
| account_state | TINYINT | YES | NULL | 1新增 2正常 3封禁 4导出 5解绑;NULL=未上报 |
| login_state | TINYINT | YES | NULL | 1在线 2离线;NULL=未上报 |
| risk_status | TINYINT | YES | NULL | 1未风控 2风控中 3待解除;NULL=未上报 |
| risk_end_time | BIGINT | YES | NULL | 风控倒计时终点(epoch毫秒) |
| cooldown_until | BIGINT | YES | NULL | 冷却到期(epoch毫秒) |
| mute_status | TINYINT | YES | NULL | 1禁言6h 2禁言24h;NULL=未上报 |
| block_error_code | VARCHAR(32) | YES | NULL | 封号错误码(401/403/440) |
| block_reason | VARCHAR(255) | YES | NULL | 封号原因(落库前按列宽截断) |
| state_source | VARCHAR(64) | YES | NULL | 状态来源前缀 NEED_REAUTH/PROXY_FAILED(截断) |
| last_state_sync_time | BIGINT | YES | NULL | 最后对账时间(epoch毫秒) |
| invalidated_at | BIGINT | YES | NULL | 失效时间(epoch毫秒;导出/解绑) |
| truth_ip | VARCHAR(45) | YES | NULL | 真实出口公网IP(上线探测;≠ip_proxy.host网关) |
| proxy_country | VARCHAR(64) | YES | NULL | 出口国家 |
| proxy_failure_count | INT | NO | 0 | 代理失败计数 |
| pull_into_group_count | INT | NO | 0 | 拉人数量 |
| created_at | BIGINT | NO | - | 创建时间(epoch毫秒) |
| updated_at | BIGINT | NO | - | 更新时间(epoch毫秒) |

**Indexes:**

| Name | Columns | Type |
|------|---------|------|
| PRIMARY | id | PRIMARY |
| uq_tenant_account | tenant_id, account_id | UNIQUE |
| idx_tenant_login | tenant_id, login_state | INDEX |
| idx_tenant_state | tenant_id, account_state | INDEX |
| idx_tenant_risk | tenant_id, risk_status | INDEX |

### account_import_batch（账号导入批次）

| Column | Type | Nullable | Default | Description |
|--------|------|----------|---------|-------------|
| id | BIGINT | NO | AUTO_INCREMENT | 主键 |
| tenant_id | BIGINT | NO | - | 租户ID |
| account_group_id | BIGINT | NO | - | 导入目标分组(→account_group.id) |
| batch_name | VARCHAR(255) | NO | - | 批次/任务名 |
| source_file_name | VARCHAR(255) | YES | NULL | 上传文件原名;纯文本导入=NULL |
| import_format | TINYINT | NO | - | 导入格式:1六段 2JSON 3全参 |
| device_os | TINYINT | YES | NULL | 机型:1安卓 2苹果 |
| account_type | TINYINT | YES | NULL | 账号类型:1个人 2商业 |
| ip_region | VARCHAR(64) | YES | NULL | 导入时选的IP国家 |
| total_rows | INT | NO | 0 | 解析总行数 |
| imported_rows | INT | NO | 0 | 成功入库行数 |
| duplicate_rows | INT | NO | 0 | 重复行数(批内/库内) |
| format_error_rows | INT | NO | 0 | 格式/凭据不全行数 |
| login_success | INT | YES | NULL | 登录成功(step1 NULL=未登录) |
| login_failed | INT | YES | NULL | 登录失败(step1 NULL) |
| login_abnormal | INT | YES | NULL | 登录异常密钥/封号(step1 NULL) |
| status | TINYINT | NO | 2 | 批次状态:1进行中 2已完成(step1同步导入即2) |
| created_at | BIGINT | NO | - | 导入时间(epoch毫秒) |
| created_by | BIGINT | YES | NULL | 创建人user_id |
| deleted_at | BIGINT | YES | NULL | 软删时间(epoch毫秒) |

**Indexes:**

| Name | Columns | Type |
|------|---------|------|
| PRIMARY | id | PRIMARY |
| idx_tenant_group | tenant_id, account_group_id | INDEX |
| idx_tenant_created | tenant_id, created_at | INDEX |

### account_import_detail（账号导入明细）

| Column | Type | Nullable | Default | Description |
|--------|------|----------|---------|-------------|
| id | BIGINT | NO | AUTO_INCREMENT | 主键 |

| batch_id | BIGINT | NO | - | 所属批次(→account_import_batch.id) |
| line_no | INT | NO | - | 行号 |
| ws_phone | VARCHAR(32) | YES | NULL | 该行WA号 |

| created_at | BIGINT | NO | - | 创建时间(epoch毫秒) |

**Indexes:**

| Name | Columns | Type |
|------|---------|------|
| PRIMARY | id | PRIMARY |
| idx_tenant_batch | tenant_id, batch_id | INDEX |
| idx_tenant_batch_result | tenant_id, batch_id, parse_result | INDEX |
